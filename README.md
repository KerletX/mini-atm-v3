# mini-atm-v3

Fonksiyonel programlama ile yazılmış
bir komut satırı ATM simülasyonudur.

Özellikler:
- Şifre kontrollü giriş
- Para yatırma / çekme
- Bakiye takibi
- İşlem geçmişi loglama
- Dosyaya veri kaydetme
- Hata yakalama (try/except)

Bu sürüm fonksiyon tabanlıdır.
Sonraki sürümlerde OOP mimarisine geçilmiştir.
